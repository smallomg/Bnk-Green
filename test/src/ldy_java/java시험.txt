1 클래스를 정의하시오
	클래스는 내가 정의할수 있는 자료형
	클래스에는 변수와 함수로 존재한다
	클래스 안에 있는 변수,함수를 멤버변수,멤버함수라고 한다
	클래스와 클래스는 서로 연결할때 객체를 생성해야한다
	객체(인스턴스)란 클래스와 다른 클래스를 연결하기 위한 변수명이다
	클래스명 객체명(인스턴스) = new 클래스명();
	이런식으로 객체를 생성하고 생성되는 과정을 생성자라고 한다
	
2 배열을 정의하시오
	동일한 같은 타입
	배열이란 변수명하나의 여러개의 공간을 나눠 내가 원하는 값을 넣는 작업이다  
	변수명 하나로 편하게 작업 가능하다
	자료형 변수명[] = new 자료형[나눌공간의 개수];

3 참조변수를 정의하시오
	내가 생각하는 참조변수란 주소값을 가리키는 변수 
	값을 가지고 있지는 않으나 주소값을 가지고있어 
	그 주소값에 맞는 배열,객체 등을 가리키고있다.
